Game Name: 
Bulls & Cows

Description: 
Computer picks a random word. User has to guess the word. User only has a set amount of guesses. 
When user makes a guess, number of bulls (correct letter/s in the correct place) and cows (correct letter/s in the wrong place) are returned.

Inputs:
- User input guesses.
- User is asked if they want to play again when game is over.

Outputs:
- ASCII art.
- Number of correct letters in the users guess.
- Basic cout.
